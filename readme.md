# EC_shop
ECショップサイト

## 環境

- PHP7.3
- MySQL5.7
- Laravel6.0

---

## 機能

- Authログイン
- カートの追加・削除機能
- 購入機能（stripe使用予定）
- 購入時のメール送信（開発環境のみ。Mailtrap使用）
